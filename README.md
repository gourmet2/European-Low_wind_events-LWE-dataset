# European-Low_wind_events-LWE-dataset
Tools for detecting, classifying, and analysing European low-wind events (LWEs) from ERA5 reanalysis data.

# ---------------------

This repository contains Python code used to construct, analyse, and visualise a European low-wind event (LWE) dataset derived from ERA5 reanalysis data. The workflow includes event detection, shallow/deep LWE classification, event merging, statistical analysis, and generation of gridded LWE flag datasets for the period 1979–2024.

The dataset is designed to support research on wind-energy variability, wind droughts, atmospheric boundary-layer processes, and large-scale circulation impacts on low-wind conditions.
